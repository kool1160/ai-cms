## V1 Success Condition

V1 succeeds if one ESP32-based soil moisture sensor can:
- Read moisture values reliably
- Display DRY / WATCH / GOOD / WET status
- Serve a usable local dashboard
- Store recent readings in runtime memory
- Display recent readings as a simple graph
- Operate as a basic outdoor prototype

- No database persistence
- No cloud connectivity
- No AI analytics

- Prototype enclosure only
